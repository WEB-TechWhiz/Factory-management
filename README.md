# 🏭 Industrial Factory Management Server

A backend system built with **Node.js, Express, and Mongoose** to schedule, monitor, and manage factory operations. The system tracks factory locations, worker shifts, and machinery statuses.

---

## ⚡ Core Features
* **Factory Registration**: Create and manage separate factory locations.
* **Worker Allocations**: Map workers to specific factories with distinct operational roles.
* **Machine State Tracking**: Keep track of machine availability and operational status.
* **Secure Sessions**: Authentication using **JWT** and secure cookies.

---

## 🛠️ Tech Stack
* **Core**: Node.js, Express.js (ESModules format)
* **Database**: MongoDB (Mongoose schemas)
* **Security**: Bcrypt.js, JsonWebToken, Cookie-parser
